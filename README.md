# exam-helper-auto-update（全员答题助手）
适配30题考试的半自动答题脚本，支持 GitHub 在线更新、自动交卷、视频题特殊模式。

## 🔧 前置要求
- 苹果商店下载 stay
- 需使用已授权的身份证号激活（与考试页面身份证号一致）。

## 🚀 安装步骤
1. 打开stay软件点右上角➕选择通过链接🔗下载
2. 点击下方链接自动跳转安装：  
   [👉 安装答题助手](https://raw.githubusercontent.com/wfxx2026/exam-helper-auto-update/main/main.user.js)
3. 页面弹出授权窗口，输入身份证号完成激活即可使用。

## 📌 核心功能
- 自动答题：0.1秒/题，适配30题全题型（单选/多选/判断）；
- 自动更新：脚本加载后3秒自动检测新版本，支持手动触发（「检查更新」按钮）；
- 自动交卷：答完30题后自动提交+结束考试，支持快捷键开关；
- 视频题模式：检测到视频页面自动调整控件透明度，不影响观看。

## 🔄 更新说明
- 自动检测：脚本启动后自动对比 `update.json` 配置，发现新版本弹窗提示；
- 手动检测：点击页面右侧「🔍 检查更新」按钮，立即触发版本校验；
- 跳过机制：选择「稍后再说」后，24小时内不再提示同一版本更新；
- 强制更新：关键修复时启用，必须升级才能继续使用。

## ⚠️ 注意事项
1. 仅用于模拟考试/练习场景，请勿用于正式考试（遵守相关规定）；
2. 身份证号需与考试页面绑定的账号一致，否则无法激活；
3. 若更新检测失败，检查网络连接或手动访问安装链接重新安装。
